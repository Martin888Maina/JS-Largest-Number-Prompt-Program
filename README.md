# Largest Number Finder

A modern, interactive web application that compares three numbers and identifies the largest value. Built with clean JavaScript, featuring an elegant user interface with smooth animations and responsive design.

## Overview

This application demonstrates core programming concepts including conditional logic, input validation, and DOM manipulation, all wrapped in a polished, user-friendly interface. The project showcases modern frontend development practices with semantic HTML, modular CSS architecture, and well-structured JavaScript.

## Features

- **Interactive Form Interface** - Clean, intuitive input fields with real-time validation
- **Smart Number Comparison** - Efficient algorithm to determine the largest of three numbers
- **Visual Feedback** - Animated result display with success and error states
- **Input Validation** - Comprehensive error handling for empty fields and invalid inputs
- **Responsive Design** - Fully optimized for desktop, tablet, and mobile devices
- **Modern Aesthetics** - Glassmorphism effects, gradient backgrounds, and smooth transitions
- **Accessible UI** - Semantic HTML structure with proper labels and ARIA attributes

## Technologies Used

- **HTML5** - Semantic markup for better accessibility and SEO
- **CSS3** - Modern styling with custom properties, flexbox, and animations
- **JavaScript (ES6)** - Event-driven programming with clean, modular code
- **Google Fonts** - Inter font family for professional typography

## Installation

1. Clone the repository to your local machine:
```bash
git clone https://github.com/Martin888Maina/JS-Largest-Number-Prompt-Program.git
```

2. Navigate to the project directory:
```bash
cd JS-Largest-Number-Prompt-Program
```

3. Open `Index.html` in your preferred web browser:
```bash
start Index.html
```

No additional dependencies or build steps required - it's a pure vanilla JavaScript application.

## Usage

1. Open the application in your web browser
2. Enter three numbers in the input fields provided
3. Click the "Find Largest Number" button
4. The application will display the largest number with a smooth animation
5. If any field is empty or contains invalid data, you'll receive a helpful error message

## Technical Highlights

### Clean Code Architecture

The JavaScript is organized into small, focused functions with clear responsibilities:
- `handleFormSubmit()` - Processes form submission
- `validateInputs()` - Ensures all fields contain valid data
- `findLargestNumber()` - Determines the maximum value using Math.max()
- `displayResult()` - Updates the UI with animated feedback

### Modern CSS Features

The stylesheet leverages contemporary CSS techniques:
- CSS custom properties for consistent theming
- Flexbox for responsive layouts
- Keyframe animations for smooth transitions
- Glassmorphism effects with backdrop-filter
- Mobile-first responsive design with media queries

### User Experience Focus

Every interaction is designed for clarity and delight:
- Real-time input validation clears previous results
- Smooth animations provide visual feedback
- Clear error messages guide users to correct input
- Accessible color contrast meets WCAG standards
